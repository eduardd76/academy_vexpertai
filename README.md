## 🚀 Quick Start

### Local Development

1. **Clone or download the repository**
   ```bash
   git clone [repository-url]
   cd ai-networking-security-academy
   ```

2. **Open in browser**
   ```bash
   # Simple: Open index.html directly
   open index.html

   # Or use a local server (recommended)
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

3. **Start learning!**
   - Browse chapters in the sidebar
   - Read theory on the left
   - Practice with code on the right
   - Track your progress

## 🌐 Deploy to Netlify

### Option 1: Drag & Drop

1. Visit [Netlify Drop](https://app.netlify.com/drop)
2. Drag the `ai-networking-security-academy` folder
3. Your site is live!

### Option 2: Git Deploy

1. Push to GitHub/GitLab/Bitbucket
2. Connect to Netlify
3. Deploy with one click

### Option 3: Netlify CLI

```bash
npm install -g netlify-cli
netlify login
cd ai-networking-security-academy
netlify deploy --prod
```

## 📝 Customization

### Adding New Chapters

Edit `js/modules.js`:

```javascript
{
    id: 'your-chapter-id',
    title: 'Your Chapter Title',
    volume: 'Volume X: Category',
    icon: 'fa-icon-name',
    theory: `# Markdown content here`,
    code: `# Python code here`,
    examples: [...],
    hint: 'Helpful tip'
}
```

### Changing Theme Colors

Edit `css/styles.css`:

```css
:root {
    --accent-primary: #58a6ff;  /* Your color */
    --bg-primary: #0d1117;      /* Background */
}
```
